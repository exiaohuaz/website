Synthetic Data Research

Edge Computing @ CMU Living Edge Lab

May 2021 - Present

Using synthetically generated image data to train object detection and classification models may bypass the typical arduous procedure of taking and annotating images.

Using a 3D model building kit rendered in Unity, I am testing how Unity3D's Perception package fairs against more complex objects, how augmenting real data with synthetic data may improve overall performance, and how tailor-making a realistic virtual environment for synthetic data affects its viability as a standalone data source.

Cache Simulator

CS:APP Labs for 15213

Spring 2021

In the memory hierarchy, the cheaper and more capacity a level has, the slower it is. Similarly, the faster the memory, the more expensive and smaller it is. Caches try to maximize the number of accesses to fast memory and minimize the accesses to slower memory by storing information that is likely to be used again. The principle of spatial and temporal locality states that a processor tends to access memory it has recently accessed (e.g. loops) and memory near memory it has accessed (e.g. arrays).

I implemented a cache simulator that records cache hits, misses, evictions, and dirty bytes given a trace file and created data structures to represent cache sets, lines, and blocks. Used least-recently-used (LRU) eviction policy for maintaining temporal locality and middle-bit indexing for spatial locality.
